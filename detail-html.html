HTML

HTML (HyperText Markup Language), yang merupakan dasar dari semua halaman web. HTML memungkinkan kita untuk menciptakan struktur dan konten pada halaman web, seperti teks, gambar, tautan, daftar, tabel, dan formulir.Dengan memahami HTML, Anda akan mampu membangun kerangka situs web yang fungsional dan informatif.

A. Mengenal Elemen dan Tag HTML

Apa Itu Elemen dan Tag HTML?

  • Elemen HTML adalah adalah komponen dasar dari halaman web yang terdiri dari tag pembuka, konten, dan tag penutup.
    Elemen ini membentuk struktur dari dokumen HTML.
  • Tag HTML adalah perintah yang memberi tahu browser bagaimana menampilkan konten. Tag ditulis dalam tanda kurung sudut (< >)
    dan biasanya hadir berpasangan. seperti (<p>) untuk memulai paragraf dan (</p>) untuk mengakhirinya.

Contoh :

                    
<p>Ini adalah paragraf </p>

Struktur Dasar Dokumen HTML

Dokumen HTML memiliki struktur dasar yang terdiri dari beberapa elemen kunci. Berikut adalah contoh sederhana :

                    
<!DOCTYPE html>

                    
<html>

                    
<head>

                        
<title>Judul Halaman</title>

                    
</head>

                    
<body>

                        
<h1>Selamat Datang di Situs Web Saya</h1>

                        
<p>Ini adalah paragraf pertama.</p>

                    
</body>

                    
</html>

Mari kita uraikan elemen-elemen penting dalam struktur ini :

  • <!DOCTYPE html>: Deklarasi ini memberitahu browser bahwa dokumen ini menggunakan HTML5.
  • <html>: Elemen root yang membungkus semua konten halaman.
  • <head>: Bagian ini mengandung informasi meta seperti judul dokumen (<title>) dan link ke file CSS.
  • <body>: Bagian yang memuat konten utama yang akan ditampilkan di halaman web.

B. Mengelola Teks dalam HTML

Teks adalah bagian penting dari setiap halaman web, dan HTML menyediakan berbagai tag untuk mengatur dan menata teks dengan cara yang sesuai.

Tag-Tag Teks dalam HTML

  • Heading Tags (<h1> - <h6>) :Digunakan untuk membuat judul dengan berbagai tingkat, dari (<h1>) yang terbesar hingga (<h6>) yang terkecil.
  • Paragraf (<p>) :Digunakan untuk membuat blok teks atau paragraf.
  • Garis Horizontal (<hr>) :Digunakan untuk memisahkan konten dengan garis horizontal.
  • Line Break (<br>) :Digunakan untuk memulai baris baru tanpa memulai paragraf baru.
  • Teks Tebal (<strong>) dan Miring (<em>) :Digunakan untuk menekankan teks, di mana (<strong>) menebalkan teks dan (<em>) memiringkan teks.
  • Quotation (<blockquote>) : Digunakan untuk menampilkan kutipan dalam bentuk blok.

Contoh :

                    
<h1>Judul utama</h1>

                    
<p>Ini adalah sebuah <strong>paragraf</strong> dengan teks <em>miring</em> dan <strong>tebal<strong>.</p>

                    
<blockquote>Ini adalah kutipan.</blockquote>

Dengan menggunakan berbagai tag ini, Anda dapat membuat halaman web yang lebih menarik dan terstruktur.


C. Membuat Hyperlink dan Menampilkan Gambar Membuat Hyperlink dengan Tag <a>

Hyperlink adalah bagian esensial dari web, memungkinkan pengguna untuk beralih dari satu halaman ke halaman lainnya.

  • Tag<a> : Digunakan untuk membuat tautan atau hyperlink. Atribut href digunakan untuk menentukan URL tujuan.

Contoh :

                    
<a href="url-tujuan">Kunjungi web saya.</a>

                    
contoh :
 
                    
<a href="https://www.example.com">Kunjungi web saya.</a>

Di sini, teks "Kunjungi web saya" akan menjadi tautan yang membawa pengguna ke URL yang ditentukan ketika di klik.


Menampilkan Gambar dengan Tag <img>

Gambar dapat memperindah halaman web dan memberikan informasi visual yang penting.

  • Tag<img> : Digunakan untuk menampilkan gambar. Atribut src menentukan sumber gambar, dan alt menyediakan deskripsi alternatif gambar untuk keperluan aksesibilitas.

Contoh :

                    
<img src="gambar.jpg" alt="Deskripsi gambar" >

Anda juga dapat menggabungkan gambar dengan hyperlink:

                    
<a href="https://www.google.com"><img src="google.jpg" alt="gambar google"></a>

Di sini, gambar berfungsi sebagai tautan ke Google.


D. Membuat Daftar dengan HTML Jenis-Jenis Daftar

HTML memungkinkan Anda untuk membuat daftar terstruktur dengan dua tipe utama :

  • Ordered List(<ol>) : Daftar berurutan, biasanya ditampilkan dengan angka.
  • Unordered List(<ul>) : Daftar tidak berurutan, biasanya ditampilkan dengan bullet points.
  • List Item(<li>) : Setiap item dalam daftar, baik di dalam (<ol>) maupun (<ul>).

Contoh :

                    
<ol>

                        
<li>Item pertama<li>

                        
<li>Item Kedua<li>

                    
</ol>

                    
<ul>

                        
<li>Item pertama<li>

                        
<li>Item Kedua<li>

                    
</ul>

E. Membuat Tabel dengan HTML

Tabel adalah cara yang efektif untuk menyajikan data dalam format yang terstruktur.


Struktur Tabel dalam HTML

  • Tag <table> : Digunakan untukmembungkus dalam pembuatan tabel.
  • Tag <tr> (Table Row): Digunakan untuk membuat baris dalam tabel.
  • Tag <th> (Table Header): Digunakan untuk membuat sel header tabel.
  • Tag <td> (Table Data): Digunakan untuk membuat sel data tabel.

Contoh :

                    
<table border="1">

                    
<tr>

                        
<th>Nama</th>

                        
<th>Umur</th>

                    
</tr>

                    
<tr>

                        
<td>Alice</td>

                        
<td>20</td>

                    
</tr>

                    
<tr>

                        
<td>Bob</td>

                        
<td>24</td>

                    
</tr>

                    
</table>

Tabel di atas akan menampilkan dua kolom (Nama dan Umur) dengan data untuk masing-masing kolom.


F. Membuat Formulir dengan HTML

Formulir adalah elemen penting dalam web, memungkinkan pengguna untuk memasukkan dan mengirimkan data. HTML menyediakan berbagai elemen untuk membuat formulir yang interaktif dan fungsional.


Struktur Formulir dalam HTML

  • Tag <form> : Elemen utama untuk membuat formulir. Atribut action menentukan URL tujuan data akan dikirim, sedangkan method menentukan metode pengiriman (GET atau POST).

Elemen Formulir yang Umum Digunakan :

  1. Input Field (<input>) :
    • Teks : Digunakan untuk memasukkan teks biasa.
      •                             
      <input type="text" name="nama">
    • Email : Digunakan untuk memasukkan alamat email.
      •                             
      <input type="email" name="email">
    • Password : Digunakan untuk memasukkan kata sandi.
      •                             
      <input type="password" name="password">
    • Radio Button : Pilihan tunggal dari beberapa opsi.
      •                             
      <input type="radio" name="gender" value="male"> Male
      
                            
      <input  type="radio" name="gender" value="female"> Female
    • Checkbox : Pilihan ganda.
      •                             
      <input type="checkbox" name="agreement"> I agree
  2. Textarea (<textarea>)
    • Digunakan untuk memasukkan teks dalam jumlah besar, seperti komentar atau pesan.
      •                             <textarea name="pesan"></textarea>
  3. Dropdown List (<select> dan <option>).
    • Digunakan untuk memilih salah satu dari beberapa opsi.
      •                                 
      <select name="kota">
      
                                    
      <option value="jakarta">Jakarta</option>
      
                                    
      <option value="bandung">Bandung</option>
      
                                    
      <option value="surabaya">Surabaya</option>
      
                                
      </select>
  4. Button
    • Submit Button (<input type="submit"> atau <button>) : Mengirimkan data formulir.
      •                                 
      <input type="submit" value="Kirim">
    • Reset Button (<input type="reset">) : Mengembalikan semua input ke keadaan awal. 
                                          
      <input type="reset" value="reset">
